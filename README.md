# HddSaver
Cookie clicker mod to auto backup to the users D:/ drive it makes a folder "D:\CookieBackups"

Just place the file in \Cookie Clicker\ and run the python or application, it will patch the start.js for the needed api addition and install the mod to the users local mods directory!

The application is simply the python file named ModInstaller packed with pyinstaller, for clean and pythonless installation, if you'd like to be sure if the authenticity the user can always use the source code directly!
